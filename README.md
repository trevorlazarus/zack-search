# Zack [zack-search]

A HTML5 only component which allows to interactively access Archival Data on a SPARQL endpoint.


## Development

* Install dependencies.

  `npm install`

* Build project.

  `npm run build`

* Run localy with proxy to SPARQL endpoint. See for current endpoint in package.json

  `npm start`
